3.  **AppCleaner** 功能在 SD Maid Pro 版本中可用，它为您的应用程序执行相同的任务。
4.  **数据库**管理功能帮助管理和优化任何正在使用的应用程序数据库。

除了上面提到的功能，它还提供了用于存储分析的工具，可以查找和删除较大的文件以及大量的应用程序删除。
