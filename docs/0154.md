# 安卓平板电脑

> 原文:[https://www.javatpoint.com/android-tablet](https://www.javatpoint.com/android-tablet)

安卓平板电脑是一款平板电脑大小的电脑，运行在谷歌的安卓操作系统上。它几乎包含了普通平板电脑的所有基本功能，包括游戏、网络浏览器、办公应用等。

由于安卓平板电脑是由谷歌设计的，谷歌基于云的应用程序也集成到安卓平板电脑中。

一些安卓平板电脑及其规格描述如下:
