### Chrome 手机有广告位吗

尽管有几个应用程序承诺在安卓 Chrome 上作为广告拦截器可以正常工作，但是它们可能不会很好地工作。但是你可以选择轻松关闭弹出广告。要禁用 chrome 浏览器上的弹出广告，您必须在安卓上按照以下步骤操作:

1.  在安卓设备上打开你的 Chrome 浏览器应用。
2.  点击地址栏右侧菜单上的三点，向下滚动至**“设置**”。前往**网站设置**。
3.  打开或关闭**弹出窗口下的开关，并重定向**。
